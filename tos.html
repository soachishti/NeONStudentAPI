How it works!

This application is an interface of the API written in Node.JS and deployed on OpenShift server. This app generate request to that server which is further full filled by scraping actual NeON Student site by that server. Server keep track of sessions to fulfil further requests.

Source Code NodeJS
https://github.com/soachishti/NeonStudentAPI

Main API Server
http://neonapi-fastnustudent.rhcloud.com/

Privacy Policy

To keep your session live we send a "token" in url request to keep session secure and to serve the requests.
After completing request this app (locally on your device) keep the result (your information) to speed up the experience and to support offline usage.
You can delete the data by going to Menu -> Setting -> Clear History and Log Out

We don't save password on API Server we delete it after logging in to NeON. For detail review routes/login.js in source code (line no ~64). For further request to NeON we keep NeON's session id on API server and these details are encrypted using AES 256 for detail review db.js in source code (line no ~45, ~62 and ~81)
